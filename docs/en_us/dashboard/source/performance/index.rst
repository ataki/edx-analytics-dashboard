#############################
Student Performance
#############################

To assess how students are doing in your course, you make a selection from the
**Performance** menu. EdX Insights provides aggregated performance data for
assignments and problems, and gives you access to the answers that students
submit for assigned questions. 

Student performance data is available in edX
Insights for problem components of these types:

* Checkboxes

* Dropdown

* Multiple choice

* Numerical input

* Text input

* Math expression input

Student submissions are updated every day. The computations include responses
received through the end of the previous day (23:59 UTC).
